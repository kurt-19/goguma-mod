# Brand and asset policy

The source code in this repository is released under AGPLv3 where applicable.

However, the AGPL-covered code license does not automatically grant permission to use official branding, logos, icons, store screenshots, service names, trademarks, hosted services, private backend infrastructure, or private credentials.

## Applies to

This policy applies to the official branding and non-code assets associated with IRC mobile app, including where applicable:

- app name used for the official hosted service;
- logos and icons;
- store screenshots and promotional graphics;
- service names and domains;
- private backend/server infrastructure;
- private credentials and production configuration.

## Forks and redistributions

Forks and redistributions should:

- rename the app;
- replace icons and store assets;
- use their own backend/server infrastructure;
- use their own API keys and TURN/STUN credentials;
- avoid implying endorsement by Goguma upstream authors or by the official IRC mobile app service.

## Code rights preserved

This policy is not intended to restrict rights granted under AGPLv3 for AGPL-covered code.
